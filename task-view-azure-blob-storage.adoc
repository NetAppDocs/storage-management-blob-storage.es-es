= Ver sus cuentas de almacenamiento de blobs de Azure
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
Después de instalar un agente de consola en Azure, la consola de NetApp puede descubrir automáticamente información sobre las cuentas de almacenamiento de Azure que residen en las suscripciones de Azure donde está instalado el agente de consola.  Se agrega un sistema Azure Blob a la página Sistemas para que pueda ver esta información.

Puede ver detalles sobre sus cuentas de almacenamiento de Azure, incluida la ubicación, el grupo de recursos, la capacidad utilizada y más.  Estas cuentas se pueden utilizar como destinos para operaciones de NetApp Backup and Recovery, NetApp Copy and Sync y NetApp Cloud Tiering.

.Pasos
. https://docs.netapp.com/us-en/bluexp-setup-admin/task-quick-start-connector-azure.html["Instalar un agente de consola"^]en la cuenta de Azure donde desea ver sus cuentas de almacenamiento de blobs de Azure.
. Desde el menú de navegación, seleccione *Almacenamiento > Sistemas*.
+
Poco después deberías ver automáticamente un sistema de almacenamiento de blobs de Azure.

+
image:screenshot-azure-blob-we.png["Una captura de pantalla de un sistema de almacenamiento de blobs de Azure."]

. Seleccione el sistema. Si desea habilitar los servicios de datos de NetApp con su almacenamiento de blobs de Azure, seleccione una acción en el panel derecho
+
link:task-blob-enable-data-services.html["Vea qué servicios de datos están disponibles y aprenda a usarlos con sus cuentas de Azure Blob Storage"] .

. Seleccione *Ingresar al sistema* o haga doble clic en el sistema para ver detalles sobre las cuentas de almacenamiento de Azure.
+
image:screenshot-azure-blob-details.png["Una captura de pantalla que muestra los detalles de un sistema Azure Blob: la cantidad total de cuentas de almacenamiento, la cantidad total de ubicaciones, los grupos de recursos, la capacidad total y más."]

+
Además de las columnas que aparecen de forma predeterminada, puede desplazarse por la página para ver propiedades adicionales de la cuenta de almacenamiento; por ejemplo, la cantidad de etiquetas y el tipo de cifrado que se ha habilitado.



.Próximos pasos
Puede cambiar algunas configuraciones de la cuenta de almacenamiento directamente desde la consola seleccionandoimage:button-horizontal-more.gif["el botón de tres puntos más"] para una cuenta de almacenamiento. link:task-change-blob-storage-settings.html["Haga clic aquí para más detalles"] .

Puede agregar nuevos depósitos seleccionando *Agregar cuenta de almacenamiento*. link:task-add-blob-storage.html["Haga clic aquí para más detalles"] .
